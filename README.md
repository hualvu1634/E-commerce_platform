E-commerce Project
Công nghệ sử dụng
Backend: Java 21, Spring Boot 3.5.9, Spring Security, JWT (0.13.0), Spring Data JPA, MapStruct 1.6.3, Lombok, Maven.

Frontend: React.

Database: MySQL 8.0.

DevOps & Công cụ: Docker, Docker Compose, Ngrok, Sepay.

Cách khởi chạy
Yêu cầu: Máy tính đã cài đặt sẵn Docker và Node.js.

1. Khởi chạy Backend & Database:
Tại thư mục gốc của dự án, chạy lệnh:

Bash
docker-compose up -d
2. Khởi chạy Frontend:
Mở terminal mới và chạy:

Bash
cd frontend
npm install
npm run dev
